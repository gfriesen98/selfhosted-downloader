About

This is a self hosted webapp designed to act as a front end of various command line applications

The main goal of this project is to utilize as little 3rd party packages as possible

  • yt-dlp
    • Download youtube videos and convert videos to mp3
    • Playlist support
    • Built in metadata editor
  • libheif
    • Convert HEIC images to jpeg or png
  • ffmpeg
    • Not Finished

Written in NodeJS/HTML/Javascript with the Express framework

See the github repository for more information
